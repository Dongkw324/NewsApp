# NewsApp

***
### 앱 설명

뉴스들을 읽고 개인적으로 중요하다 싶은 뉴스들을 저장할 수 있는 어플. 또한 더 이상 필요없다 싶으면 삭제도 가능하다.

***
### 기술 스택

**Kotlin**</br>
**Dagger-hilt**</br>
**Retrofit2**</br>
**Room**</br>
**ViewBinding**</br>
**Coroutine**</br>

***
### 뉴스 API

[News Api](https://newsapi.org/)에 접속하여 개인 API Key를 발급받을 수 있다.
